# Glosario

## RBAC
Método de seguridad que restringe el acceso a redes o sistemas según el rol de cada usuario, en lugar de asignar permisos individualmente.

## SAS Tokens
Es una cadena segura que se añade a una URL para otorgar acceso delegado, limitado y temporal a recursos de almacenamiento (blobs, archivos, tablas, colas) sin exponer las claves de la cuenta.

## Standard LRS
Es una opción de almacenamiento de datos en la nube, principalmente utilizada en Azure Storage, que equilibra un costo bajo con una alta durabilidad y disponibilidad dentro de un solo centro de datos.
